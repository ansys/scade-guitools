# Example of property page

## Pre-requisites
This example requires Ansys SCADE 2024 R2 or greater.

## Setup
* Register the package to SCADE as detailed in
  [Install in user mode](<https://guitools.scade.docs.pyansys.com/contributing.html#install-in-user-mode>).
* Install the `guitools-examples-page` example package so that
  it is accessible in a Ansys SCADE environment.

  You must use a Python interpreter compatible with your version of Ansys SCADE,
  for example Python 3.10 for Ansys SCADE 2024 or 2025 releases.

  ```cmd
  python -m pip install --user .
  ```

## Usage
* Run Ansys SCADE
* Open a SCADE model
* Select an operator
* Activate the *GUI Tools Example* tab of the Properties window
