





:class:`RadioBox`
=================

.. py:class:: ansys.scade.guitools.control.RadioBox(owner, buttons: List[Tuple[str, str]], x: int, y: int, w: int, text: str = '', h: int = c.GROUP_RADIO_BOX_HEIGHT)

   Bases: :py:obj:`scade.tool.suite.gui.widgets.GroupBox`


   
   Defines a bundle made of a group and a set of radio button controls.

   The group is hidden when text is empty.

   :Parameters:

       **owner** : :obj:`Any`
           owner of the control

       **buttons** : :class:`python:list`\[:class:`python:tuple`\[:class:`python:str`, :class:`python:str`]]
           Descriptions of the buttons: value and text associated to the buttons.

       **x** : :class:`python:int`
           Horizontal position of the control.

       **y** : :class:`python:int`
           Vertical position of the control.

       **w** : :class:`python:int`
           Width of the control.

       **h** : :class:`python:int`
           Height of the control, default 0

       **text** : :class:`python:str`
           Text of the group control, default empty.
