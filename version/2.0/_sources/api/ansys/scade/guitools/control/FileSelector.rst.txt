       **extension** : :class:`python:str`

       **directory** : :class:`python:str`
           Initial directory of the file selector dialog box, or the current directory when empty.

       **filter** : :class:`python:str`

       **wl** : :class:`python:int`

       **h** : :class:`python:int`
           Height of the control, default csts.EDIT_HEIGHT.

       **reference** : :class:`python:str`
